# Aneurasync AI Executive Team — 役職・責務一覧

## Chief of Staff（AI Executive Office）

**役割**: CEO の右腕。全部門の統括・優先順位管理・意思決定補佐。

| 責務 | 詳細 |
|------|------|
| 日次統括 | 各部門の進捗を集約し CEO にブリーフィング |
| 優先順位管理 | `docs/weekly-priorities.md` の更新提案 |
| 意思決定支援 | 判断材料の整理、トレードオフの明示 |
| 部門間調整 | 部門横断の課題を検知し解決策を提案 |
| 承認フロー管理 | CEO 承認が必要な事項の起票と追跡 |
