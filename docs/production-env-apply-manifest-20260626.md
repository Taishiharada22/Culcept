# PRODUCTION ENV 適用マニフェスト（2026-06-26）— 全191 env 完全分類

> 目的: 最新 /plan 体験を確実にユーザーへ。全 env を「実値SET / =true / =false・OMIT」に確定。
> 重要: local env 丸ごとコピー禁止（dev/canary/REALITY が true で混ざると prod 破壊）。本書が正本。
> 反映規則: Production scope に設定 + NEXT_PUBLIC_* は build 時 inline ゆえ設定後に redeploy 必須。
> 値は本書に書かない（secret）。CEO が実値を入れる。

## 最重要（30%しか出ていない最大原因）
plan 体験 flag が Production scope に揃っていない / 設定後に redeploy していない 可能性大。
NEXT_PUBLIC_* は build 時 inline ＝ Preview に入れても Production build に無ければ効かない。
-> 下の TIER2 を 全部 Production scope に true で入れて 1 回 redeploy が本命。

3. TIER3 は 設定しない or false(特に local から流入させない)。TIER4 は入れない。
4. redeploy(build cache off) -> NEXT_PUBLIC_* 反映。
5. P7 Phase B チェックリストで実機 + Runtime Logs 確認。
