# リサーチレポート: ファッション/ワードローブ管理アプリ競合調査

日付: 2026-03-17
担当: Research Unit
分類: 競合/トレンド調査

---

## 要約（3行以内）

ワードローブ管理アプリ市場は2026年時点で85M+ユーザー規模に成長し、AIスタイリングが標準機能化。しかし全アプリが「服の管理・コーディネート提案」に集中しており、ユーザーの深層心理・判断原理・スタイルアイデンティティを掘り下げるアプローチは皆無。Aneurasync の「第二の自己」コンセプトは明確な空白地帯に位置する。

### 1. Stylebook（iOS のみ）

**概要:** 2009年リリースの老舗ワードローブ管理アプリ。家族経営の小規模インディー開発。15年以上の実績。買い切り$4.99。

| カテゴリ | 評価 | 詳細 |
|---------|------|------|
| ワードローブ管理 | ★★★★★ | 90+機能。AI背景除去、5種類のインポート方法（AIテキスト→画像生成含む）、ドラッグ&ドロップ、Web クリッピング、一括インポート |
| AI/分析機能 | ★★★☆☆ | コスト・パー・ウェア、カラー分布、着用頻度TOP10、未使用アイテム検出。ただしAIコーデ提案はなし |
| 毎日の提案 | ★☆☆☆☆ | なし。Outfit Shuffle/Style Shuffleでランダム提案のみ |
| スタイル学習 | ★★☆☆☆ | 手動ログに基づく統計のみ。パーソナライゼーションなし |
| コスト追跡 | ★★★★★ | コスト・パー・ウェア、クローゼット総額、アイテム別追跡 |
| 天気連動 | ☆☆☆☆☆ | なし |
| ソーシャル機能 | ☆☆☆☆☆ | なし |
| アイデンティティ/自己理解 | ☆☆☆☆☆ | なし |
| オフライン対応 | ★★★★★ | 完全ローカル動作（iCloud同期はオプション） |

**強み:**
- 圧倒的な手動管理機能の深さ（90+機能）
- 買い切りモデル（$4.99で永続利用）
- マガジン風コラージュによるコーデ作成
- 体型データ・ブランド別サイズ保存
- 2025年アップデートでAI画像生成・背景除去が大幅改善

**弱み:**
- iOS限定（Android版なし）
- AI提案・天気連動・ソーシャル機能が一切ない
- 「整理する」アプリであり「決める」アプリではない
- 買い切りモデルのため開発リソースが限定的

**ユニークな差別化:** 手動コントロールの極致。「コントロール重視派のゴールドスタンダード」。

### 2. Acloset（iOS / Android）

**概要:** 韓国発のAI搭載ワードローブアプリ。400万ユーザー、100万+ダウンロード。無料（100アイテムまで）、年間30ポンド〜。

| カテゴリ | 評価 | 詳細 |
|---------|------|------|
| ワードローブ管理 | ★★★★☆ | 写真撮影・オンライン検索でアイテム追加。AI背景除去・自動タグ付け。複数クローゼット対応 |
| AI/分析機能 | ★★★★☆ | AIスタイリスト（質問応答型）、パーソナルカラー分析、骨格診断、コスト・パー・ウェア |
| 毎日の提案 | ★★★☆☆ | 天気・スケジュール連動の日替わりコーデ提案。ただし精度に問題あり |
| スタイル学習 | ★★★☆☆ | 着用データに基づく統計。AIショッピングアドバイザー（重複回避・汎用性重視の推薦） |
| コスト追跡 | ★★★★☆ | 購入日・価格記録、コスト・パー・ウェア、支出傾向分析 |
| 天気連動 | ★★★★☆ | 天気ベースのコーデ提案あり |
| ソーシャル機能 | ★★★★☆ | SNS風コーデフィード、フォロー機能、中古品マーケットプレイス |
| アイデンティティ/自己理解 | ★☆☆☆☆ | パーソナルカラー・骨格診断のみ（表層的） |
| オフライン対応 | ★★☆☆☆ | AI機能はオンライン必須 |

**強み:**
- UIの美しさ（「Best Interface」評価）
- AI背景除去・自動タグ付けの精度が高い
- マーケットプレイス内蔵（整理→売却→購入の循環）
- 無料枠でも基本機能が使える

**弱み:**
- AIコーデ提案の精度が不十分（同じアイテムの繰り返し、不自然な組み合わせ）
- 写真アップロードのバグ報告あり
- 無料は100アイテムまでの制限
- 有料プランが複数段階で分かりにくい

**ユニークな差別化:** クローゼット管理とマーケットプレイスの統合（整理→売却→購入のクローズドループ）。

### 4. Cladwell（iOS / Android）

**概要:** カプセルワードローブ哲学に基づく毎日のコーデ提案アプリ。100万+ダウンロード。無料（制限付き）、$7.99/月〜。

| カテゴリ | 評価 | 詳細 |
|---------|------|------|
| ワードローブ管理 | ★★★☆☆ | アイテム登録（色・パターン・スタイル）、カプセルワードローブ定義（30-50着） |
| AI/分析機能 | ★★★★☆ | ChatGPT連携のAIスタイリスト、着用データ分析、カラー傾向、ワードローブ利用率 |
| 毎日の提案 | ★★★★★ | 天気・クローゼット・着用頻度に基づく毎日のコーデ提案（コア機能） |
| スタイル学習 | ★★★★☆ | 着用ログから学習、スタイルプロファイル生成、均等着用のためのローテーション管理 |
| コスト追跡 | ★★★☆☆ | 基本的な追跡あり（有料プランのみ統計閲覧可） |
| 天気連動 | ★★★★☆ | 天気ベースの提案（ただし夏季の精度に問題報告あり） |
| ソーシャル機能 | ★☆☆☆☆ | 限定的 |
| アイデンティティ/自己理解 | ★☆☆☆☆ | スタイルプロファイルはあるが表層的 |
| オフライン対応 | ★★☆☆☆ | AI機能はオンライン必須 |

**強み:**
- 「毎日何を着るか」の課題に最も直接的にアプローチ
- カプセルワードローブ哲学との整合性
- ChatGPT連携による自然言語でのスタイル相談

**弱み:**

- 無料プランの厳しい制限（1日1コーデ、7日分のみログ）

**ユニークな差別化:** カプセルワードローブ哲学 + 毎日の提案に特化。「所有を減らす」ライフスタイル支援。

## インサイト

### 1. 全競合に共通する「天井」 -- アイデンティティ・自己理解の不在

全8アプリを調査した結果、**「あなたは何者か」を深掘りするアプリは1つも存在しない**。

- **Acloset** のパーソナルカラー/骨格診断は「外見の分類」であり、判断原理や価値観には触れない
- **Cladwell** のスタイルプロファイルは「何色を多く着るか」レベルの表層データ
- **Indyx** のプロスタイリストは対話を通じた気づきを提供するが、体系的な自己理解フレームワークはない
- **Good On You** は「ブランドの倫理性」を評価するが、ユーザー自身の価値観の探求ではない

ファッション心理学の学術研究（2025-2026）では、衣服選択が性格特性・自己認識・アイデンティティ構築に深く結びつくことが明らかになっているにもかかわらず、それを製品化したアプリは存在しない。

### 2. AI提案の「品質の壁」

AI コーデ提案を持つアプリ（Acloset, Cladwell, Whering）は全て、ユーザーから**提案精度の不満**を受けている。

- 同じアイテムの繰り返し提案
- 季節/天気に合わない組み合わせ
- 「なぜこの組み合わせなのか」の説明不足

これは「服のデータ」だけでAIが動いており、**ユーザーの判断基準・好みの深層構造**を理解していないことが根本原因。

### 3. 「整理」vs「決定」の分断

市場は明確に2つに分かれている:
- **整理系:** Stylebook, Smart Closet, Indyx（手動管理の極致）
- **決定系:** Cladwell, Acloset（AI提案だが精度不足）

「深く理解した上で提案する」アプリは不在。

### 4. サステナビリティは「手段」であり「目的」の空白がある

Whering, Good On You, Save Your Wardrobe は「環境に良い」を軸にしているが、**なぜその人がサステナビリティを重視するのか、その価値観の根源は何か**には踏み込まない。

### 5. ヒューマンスタイリストの限界

Indyx は「スタイルは人間のアート」として人間スタイリストを統合しているが:
- $60-$295+の高コスト
- スケーラビリティの限界
- スタイリストの質のばらつき

AIが人間スタイリストの「深い理解」の部分を再現できれば、大きな競争優位になる。

## Aneurasync の競争ポジション

### 明確な空白地帯

```
[表層] 服の管理・分類  ← Stylebook, Smart Closet, Indyx
[中層] AIコーデ提案    ← Cladwell, Acloset
[深層] 価値観・判断原理  ← 【空白】 ← Aneurasync が狙うべき層
[最深層] 第二の自己      ← 【空白】 ← Aneurasync のビジョン
```

### 競合に対する差別化ポイント

| Aneurasync の強み | 競合との違い |
|------------------|-------------|
| Stargazer（深層観測） | 判断基準・矛盾・無自覚な傾向を観測 -- 他アプリはカラー/カテゴリ統計のみ |
| 「なぜそれを選ぶのか」の解明 | AI提案の「なぜ」を説明できる唯一のアプリになれる |
| 時間経過による変化の追跡 | 他アプリは静的なプロファイル。Aneurasync は変化の法則を掴む |
| 観測型UX | アンケートではなく行動観測から深層に到達 |

## 推奨アクション

### 短期（今月中）
1. **Stargazer の深層観測を「スタイル判断」に接続する具体的なUXフローを設計** -- 競合の空白地帯を製品として形にする
2. **「なぜこのコーデなのか」を説明できるAI提案**のプロトタイプ検討 -- 競合のAI提案精度問題を逆手に取る

### 中期（1-2ヶ月）
3. **ワードローブ管理の基本機能は「十分なレベル」で実装** -- Stylebook/Indyx レベルの網羅性は不要。最低限の登録・閲覧・コーデ作成があれば良い
4. **「自分って、そういう人間だったのか」モーメントの設計** -- 競合に全く存在しない体験を作り込む

### 長期
5. **ファッション心理学の知見をプロダクトに統合** -- 学術的に裏付けのある自己理解フレームワークの構築
6. **Indyx のヒューマンスタイリストが提供する「深い対話」をAIで民主化** -- $60-$295のサービスを無料/低コストで提供
