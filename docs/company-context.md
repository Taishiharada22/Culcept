# Aneurasync Company Context

## Mission
「自分って、そういう人間だったのか」── ユーザーが自分自身の深層を発見する体験を届ける。

## Product
Aneurasync は、ファッション・スタイルを入口に、ユーザーの判断原理・揺れ方・深層心理・無自覚な内面傾向を観測し、「第二の自己」として機能するプラットフォーム。

### Core Features
| Feature | 概要 | 優先度 |
|---------|------|--------|
| **Stargazer** | **性格・判断特性の深層観測エンジン** | **最優先** |
| My Style | ワードローブ管理 + スタイル DNA | 高 |
| Body Color | パーソナルカラー・体型診断 | 高 |
| Calendar | 天気連動コーディネート提案 | 中 |
| Origin | ライフマップ | 中 |
| Battle | スタイル対決 | 中 |
| Drops | ファッションアイテム出品 | 中 |
| Rendezvous | アバター先行型マッチング | 低（今月は後回し） |

## Target User
- ファッションに興味がある 20-30 代
- 自己理解を深めたい人
- 「何を着るか」を通じて「自分は何者か」を探求したい人

## Competitive Edge
1. 表面的プロフィールではなく深層心理を掴む
2. 質問 = アンケートではなく観測の入口
3. 回答 → 仮説 → 深掘り → 本音に到達するプロセス

## CEO 方針（2026年3月）

### 最優先テーマ
Stargazer 深層観測の完成

### 今月の成功条件
1. コア機能の完成（主要機能が一通り動く）
2. 初期ユーザー獲得（テストユーザーからフィードバック）
3. 世界観の確立（体験・UI・トーンの一貫性）
4. デプロイ可能状態（本番品質）

### 今はやらないこと
- マネタイズ設計（課金・収益モデル）
- 大規模マーケティング（広告・SNS 運用・集客施策）
  - ただし、少人数の初期検証ユーザー獲得（知人・招待制）は行う

### 意思決定原則
- 迷ったらスピードより整合性と世界観を優先
- AI は提案・実行候補まで。最終決定は CEO
- 本番反映・課金変更・法務変更・対外公開は必ず CEO 承認
