# Weekly Review — 週次レビュー

Chief of Staff として週次レビューを実行してください。

## 手順

1. `docs/weekly-priorities.md` で今週の目標と達成度を確認
2. `git log --oneline --since="7 days ago"` で今週の変更一覧を作成
3. `docs/decision-log.md` で今週の意思決定を確認
4. 来週への申し送り事項を整理

## 出力フォーマット

```
## 週次レビュー [今週の日付範囲]

### 成果サマリー
- [主要な成果 3-5 個]

### 部門別ステータス
| 部門 | ステータス | 今週の成果 | 課題 |
|------|-----------|-----------|------|
| Product | 🟢/🟡/🔴 | ... | ... |
| Research | ... | ... | ... |
| Build | ... | ... | ... |
| Growth | ... | ... | ... |
| Ops | ... | ... | ... |

### 今週の意思決定
- [decision-log からの要約]

### 来週への申し送り
1. ...
2. ...
3. ...

### 数値（把握可能な範囲で）
- コミット数: ...
- 新機能: ...
- バグ修正: ...
```
