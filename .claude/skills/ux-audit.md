# UX Audit — UX・世界観の一貫性監査

Product Unit の UX Designer として、指定されたページ/機能の UX 監査を行ってください。

## 手順

1. 引数で指定されたページ（なければ直近変更のページ）のコードを読む
2. 以下の観点でチェック:

### チェック項目

**デザインシステム準拠**
- [ ] GlassCard, GlassBadge, GlassButton 等を使用しているか
- [ ] glassmorphism-design.tsx のコンポーネントを活用しているか
- [ ] 独自のスタイルがデザインシステムと競合していないか

**日本語 UI**
- [ ] ラベル・ボタン・説明文がすべて日本語か
- [ ] 英語が混在していないか（技術用語を除く）

**アニメーション**
- [ ] Framer Motion を使用しているか
- [ ] FadeInView 等の共通アニメーションを活用しているか

**世界観の一貫性**
- [ ] Aneurasync のトーン（深層観測、第二の自己）と整合しているか
- [ ] 他のページと体験の連続性があるか

**アクセシビリティ基本**
- [ ] ボタン/リンクに適切なラベルがあるか
- [ ] コントラスト比が低すぎる色の組み合わせがないか
